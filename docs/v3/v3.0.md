# V3.0 — 1 Agent, 2 Robots (Overhead Cam)

## Setup
- **Camera:** 1 overhead USB cam (480p)
- **Agent:** 1 shared brain
- **Robot:** 2× SO-ARM100 (6-DOF)

## Architecture

```
Overhead cam ──→ Encoder ──→ z (joint latent, 192-dim)
                                    │
                              Predictor (Mamba-2+Attention)
                                    │
                              ┌─────┴─────┐
                         action head 1   action head 2
                              │               │
                         Robot 1 arm    Robot 2 arm
```

- 1 predictor predicts BOTH robots' future latents
- CEM planning over joint action space (action_dim = 2 × 6 = 12)
- Cost = distance(z_predicted, z_goal) for both robots

## Limitation
1 agent = no coordination bottleneck. Both robots act as "2 hands of 1 brain". No social learning.
