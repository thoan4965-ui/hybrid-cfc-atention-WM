## 5. World Model / RL / Robot Manipulation Projects at ISEF

### 5.1 Search Result: NONE Found (2024-2026)

After searching ISEF project databases, Society for Science abstracts, and news coverage:
- **Zero ISEF projects (2024-2026) involve world models** (e.g., Dreamer, LeWM, Daydreamer-style)
- **Zero ISEF projects involve model-based reinforcement learning** for robot control
- **Zero ISEF projects involve robot manipulation** at a research level (pick-place, push, assembly)

### 5.2 Closest Related Projects Found

**2016 (historical):** Fido — Universal robot control using Q-learning + neural networks. Won 2nd place at ISEF 2016 + NASA award. Reinforcement learning with limited feedback. Still far from modern MBRL.

**2023 (ROBO):** Rohan Kalahasty — "Hybrid Plasticity: Adaptive, Brain-Like AI via Prefrontal Cortex Inspired Meta-Learning." Used OpenAI Gym to test PPO/SAC comparisons. Most RL-adjacent project found. Third Award.

**2025 (SOFT):** Ronit Dey (NC, USA) — "DISTRACT: Driver Inattention State Transition Recognition Using Attention-Based Convolutional Transformers." Uses attention + conv for time-series. 4th Award in System Software.

**2026 (not ISEF, but research context):** Recent world model papers (World-Gymnast, RISE, Boundless World Model) are all published at NeurIPS/CVPR/ICLR 2025-2026. None have percolated to high school level yet.

### 5.3 Why This Gap Exists
1. **Compute barrier:** Training world models requires GPU clusters (4-8× A100/L40S). High school students rarely have this access.
2. **Knowledge barrier:** World models combine video prediction, latent dynamics, planning — requires understanding of multiple advanced topics.
3. **Hardware barrier:** Robot manipulation requires physical robots ($5K-$50K+). Even simulation requires MuJoCo/PyBullet expertise.
4. **Time barrier:** World model projects take 6-18 months for experienced PhD students. High school timeline (1 academic year) insufficient.

## 9. Final Assessment

**ISEF is NOT a research competition at the PhD level.** It rewards:
1. Real working hardware (above all)
2. User testing with measurable outcomes
3. Clear societal impact + cost-effectiveness narrative
4. Multi-year project trajectory
5. Effective science communication

**LeWM technical advantage is real** — no ISEF project in 2024-2026 approaches the AI/ML sophistication of a world model with CfC+Attention hybrid. This is a genuine competitive moat.

**But technical depth alone does NOT win ISEF.** The winning pattern consistently shows that application + hardware + user testing + narrative matter more than algorithmic novelty.

**LeWM's highest-probability path:** Pair V1.1 benchmark numbers (simulation) with V0 real robot grasp demo + real-time CEM. Add cost comparison + impact narrative. Target ROBO category. Aim for 2nd-3rd at ISEF level, 1st at affiliate level.
