# V3.1 — 2 Agents, 2 Robots (Overhead + 2 Ego Cams)

## Setup
- **Camera:** 1 overhead + 2 ego (1 per robot)
- **Agent:** 2 agents (1 per robot), independent but coordinated
- **Robot:** 2× SO-ARM100 (6-DOF)

## Architecture

```
Overhead cam ──→ Joint Encoder ──→ z_shared
                                              │
Robot 1 ego ──→ Encoder1 ──→ z1              │
Robot 2 ego ──→ Encoder2 ──→ z2              │
                                              │
z1' = z1 + CrossAttn(Q=z1, K=z2, V=z2)       │
z2' = z2 + CrossAttn(Q=z2, K=z1, V=z1)       │
                                              ▼
z = concat(z_shared, z1', z2') ──→ Social Predictor
                                           │
                              ┌────────────┴────────────┐
                         Robot 1 predictor          Robot 2 predictor
                              │                           │
                     MDN-GMM head (K comps)      MDN-GMM head (K comps)
                              │                           │
                        NLL + KL(z||prior)          NLL + KL(z||prior)
```

## Key Components

### Cross-Attention
Multi-head cross-attention between robot latents:
```
z1' = z1 + MultiHead(Q=z1, K=z2, V=z2)   # Robot 1 attends to Robot 2
z2' = z2 + MultiHead(Q=z2, K=z1, V=z1)   # Robot 2 attends to Robot 1
```
Allows each agent to reason about the other's state when planning.

### CLIP Joint Embedding
Both ego encoders are aligned to CLIP semantic space:
- `Encoder1` and `Encoder2` outputs mapped to CLIP embedding via learned projection
- Shared CLIP space ensures `z1` and `z2` are semantically comparable
- Cross-attention quality depends on this alignment
- Training: contrastive loss between latent views of same scene

### Social Predictor (MDN-GMM)
Predicts future latents as Gaussian Mixture:
- Each robot: predict K future components (μ_k, σ²_k, π_k)
- Captures multi-modal futures: "robot 2 might go left OR right"
- Loss: NLL + λ_KL · KL(pred || prior)

### CEM Planning
Joint CEM over (action1, action2):
- Cost = distance(z_goal, z_pred) for both robots
- Collaboration implicit: both latents in shared space, cross-attn already coordinated
